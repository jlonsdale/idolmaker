■ビーム式ファーシェーダ適用方法
- MMDでMMEfectが使えるようにして下さい。
- モデルを読み込みます。
- MMEffectのサブセット展開からビキニ材質にそれぞれ対応したFar_xxxx.fxを適用して下さい。
  - 水着上 ← Far_tops.fx
  - 水着下Y ← Far_bottomY.fx (ホットパンツ版では不要です)
  - 水着下T ← Far_bottomT.fx (ホットパンツ版では不要です)
- 水着3, 4にのみ有効になります。水着1, 2では有効になりません。
