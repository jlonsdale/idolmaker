つくったひと：ビームマンP

マンです。
ファーシェーダです。もふもふです。


内容物：
Far
メインのファーシェーダです。

簡単な使い方：
モデルのもふもふさせたい材質にFar.fxを適用すればもふります。
Far.fxの中身を開いて毛の長さや量、移動量（毛並みの乱れみたいな？）を調整して良い具合にしてください。

テクスチャとかいじれる人向けの使い方：
（モデルのテクスチャと同サイズのテクスチャを作成、Far.fxのFarTexScaleを1.0にしてください）

・far_AmountMap.png 
毛の生える量を指定するテクスチャです。モデルのUVに合わせて生やしたい所に白、生やしたくない所に黒を置いてください。

・far_VectorMap.png
毛の"流れ"を指定するテクスチャです。赤、緑成分に傾き具合を指定してください。
赤：128,緑：128が"正面"を向いた状態です。

何か大変だろうけど　がんばれ！がんばれ！
