MikuMikuDanceデータ：RGBA式水着ボディC　ver.1.0

本データはMikuMikuDance用（Ver.8.03で動作確認）の水着ボディの素体モデルです。頭部は含まれていません。
MikuMikuMovingで使う場合はMikuMikuMoving用のRGBA式水着ボディCを使用してください。
